Site Report

This portfolio website was developed as part of a web development assignment. The aim of the project was to demonstrate understanding of HTML and CSS by creating a responsive multi-page website. The development process began with planning the layout and structure of the website. Four main pages were created: home, projects, contact, and site report. Each page serves a specific purpose and is connected using a navigation menu. HTML was used to structure the content, while CSS was used for styling and layout design. CSS Grid was applied in the projects page to create a responsive layout that adjusts to different screen sizes. Basic styling such as colors, spacing, and typography was used to improve user experience. One of the challenges faced during development was maintaining consistent design across all pages. This was solved by using a shared CSS file. Another challenge was making the layout responsive, which was achieved using flexible grid properties. Overall, this project helped in understanding the fundamentals of web development, including page structure, styling, and responsive design. It also improved problem-solving skills and confidence in building websites.
